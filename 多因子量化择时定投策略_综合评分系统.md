## 二、技术指标参数设定

### 2.1 RSI（相对强弱指数)

```
参数：RSI(6, 12, 24)
├── RSI(6)  → 超短期动量
├── RSI(12) → 短期动量
└── RSI(24) → 中期动量
```

**买入阈值参考**：

| 市场环境 | RSI(6) | RSI(12) | RSI(24) | 信号强度 |
|---------|--------|---------|---------|---------|
| 极度超卖 | < 15 | < 20 | < 25 | ★★★★★ |
| 严重超卖 | 15-25 | 20-30 | 25-35 | ★★★★☆ |
| 中度超卖 | 25-35 | 30-40 | 35-45 | ★★★☆☆ |
| 轻度超卖 | 35-45 | 40-50 | 45-55 | ★★☆☆☆ |
| 中性区域 | 45-55 | 50-60 | 55-65 | ★☆☆☆☆ |

**关键买入条件**：
- RSI(6) < 30 且 RSI(12) < 35 且 RSI(24) < 40 → 强烈买入信号
- RSI(6) 上穿 RSI(12) 形成金叉 → 动量转强
- RSI 底背离（价格创新低，RSI未创新低）→ 反转信号

### 2.2 KDJ（随机指标)

```
参数：KDJ(9, 3, 3)
├── K线 → 快速线
├── D线 → 慢速线
└── J线 → 方向敏感线 (J = 3K - 2D)
```

**买入阈值参考**：

| 市场环境 | K值 | D值 | J值 | 信号强度 |
|---------|-----|-----|-----|---------|
| 极度超卖 | < 10 | < 15 | < 0 | ★★★★★ |
| 严重超卖 | 10-20 | 15-25 | 0-10 | ★★★★☆ |
| 中度超卖 | 20-30 | 25-35 | 10-20 | ★★★☆ |
| 轻度超卖 | 30-40 | 35-45 | 20-30 | ★★☆☆☆ |

**关键买入条件**：
- K < 20 且 D < 30 → 超卖区域
- K 上穿 D 形成金叉 → 买入信号
- J < 0 → 极度超卖，反弹概率高
- KDJ 底背离 → 强烈反转信号

### 2.3 BIAS（乖离率)

```
参数：BIAS(n) = (收盘价 - n日均价) / n日均价 × 100%
常用：BIAS(5), BIAS(10), BIAS(20), BIAS(60)
```

**买入阈值参考**：

| 时间周期 | 轻度偏离 | 中度偏离 | 严重偏离 | 极度偏离 |
|---------|---------|---------|---------|---------|
| BIAS(5) | -2% ~ -3% | -3% ~ -5% | -5% ~ -8% | < -8% |
| BIAS(10) | -3% ~ -5% | -5% ~ -8% | -8% ~ -12% | < -12% |
| BIAS(20) | -5% ~ -8% | -8% ~ -12% | -12% ~ -18% | < -18% |
| BIAS(60) | -8% ~ -12% | -12% ~ -18% | -18% ~ -25% | < -25% |

**关键买入条件**：
- BIAS(20) < -10% 且 BIAS(60) < -15% → 中长期严重偏离，买入机会
- 短期BIAS快速下探后反弹 → 短期抄底信号

## 三、先进综合评分系统（多因子加权模型)

### 3.1 评分框架

```
总评分 = Σ(单项因子得分 × 权重) + 市场环境调整分 + 动量确认分
范围：0 ~ 100 分
```

### 3.2 因子权重分配

| 因子类别 | 具体指标 | 权重 | 说明 |
|---------|---------|------|------|
| **动量因子** | RSI(6,12,24)综合 | 25% | 反映价格动量 |
| **随机因子** | KDJ(9,3,3)综合 | 20% | 反映随机波动 |
| **偏离因子** | BIAS(5,10,20,60)综合 | 20% | 反映价格偏离度 |
| **趋势因子** | 均线系统(MA5/MA10/MA20/MA60) | 15% | 反映趋势方向 |
| **量能因子** | 成交量/成交额变化 | 10% | 反映资金情绪 |
| **波动因子** | 波动率(历史波动率/ATR) | 10% | 反映风险水平 |

### 3.3 各因子评分细则

#### 3.3.1 RSI 综合评分 (满分25分)

```
RSI评分 = RSI6_score × 0.4 + RSI12_score × 0.35 + RSI24_score × 0.25

RSI评分规则：
- RSI < 20: 25分 (极度超卖)
- 20 ≤ RSI < 30: 20分 (严重超卖)
- 30 ≤ RSI < 40: 15分 (中度超卖)
- 40 ≤ RSI < 50: 10分 (轻度超卖)
- 50 ≤ RSI < 60: 5分 (中性偏强)
- RSI ≥ 60: 0分 (偏强区域)

附加加分：
- RSI形成底背离: +3分
- RSI(6)上穿RSI(12)金叉: +2分
```

#### 3.3.2 KDJ 综合评分 (满分20分)

```
KDJ评分 = K_score × 0.3 + D_score × 0.3 + J_score × 0.4

K/D评分规则：
- < 10: 20分
- 10-20: 16分
- 20-30: 12分
- 30-40: 8分
- 40-50: 4分
- ≥ 50: 0分

J评分规则：
- J < 0: 20分
- 0 ≤ J < 10: 16分
- 10 ≤ J < 20: 12分
- 20 ≤ J < 30: 8分
- 30 ≤ J < 40: 4分
- J ≥ 40: 0分

附加加分：
- K上穿D金叉: +3分
- KDJ底背离: +3分
```

#### 3.3.3 BIAS 综合评分 (满分20分)

```
BIAS评分 = BIAS5_score × 0.2 + BIAS10_score × 0.25 + BIAS20_score × 0.3 + BIAS60_score × 0.25

BIAS评分规则（负偏离越大，分数越高）：
- < -15%: 20分
- -15% ~ -10%: 16分
- -10% ~ -7%: 12分
- -7% ~ -5%: 8分
- -5% ~ -3%: 4分
- ≥ -3%: 0分

附加加分：
- 短期BIAS从极值快速反弹: +2分
- 多周期BIAS同步负偏离: +2分
```

#### 3.3.4 趋势因子评分 (满分15分)

```
趋势评分 = 均线系统评分 + 趋势强度评分

均线评分：
- 价格 < MA60 < MA20 < MA10 < MA5: 15分 (极度空头，反转潜力大)
- 价格 < MA60, 但MA排列混乱: 10分
- 价格接近MA60下方: 8分
- 价格位于MA60-MA20之间: 5分
- 价格 > MA60: 0分

趋势强度评分：
- 长期下行趋势中: +3分
- 中期下行趋势中: +2分
- 短期下行后企稳: +1分
```

#### 3.3.5 量能因子评分 (满分10分)

```
量能评分：
- 成交量处于近期低位(低于20日均量50%): 10分 (地量见地价)
- 成交量低于20日均量30%: 8分
- 成交量低于20日均量10%: 6分
- 成交量接近20日均量: 3分
- 成交量高于20日均量: 0分

附加加分：
- 价格创新低但成交量萎缩(量价背离): +2分
- 底部出现放量阳线: +3分
```

#### 3.3.6 波动因子评分 (满分10分)

```
波动率评分：
- 波动率处于历史高位(>80%分位): 10分 (高波动后往往伴随机会)
- 波动率处于70-80%分位: 8分
- 波动率处于50-70%分位: 5分
- 波动率处于30-50%分位: 3分
- 波动率处于低位(<30%分位): 0分

ATR评分：
- ATR处于高位后回落: +2分
- ATR持续扩大中: +1分
```

### 3.4 市场环境调整分

```
市场环境调整 = 大盘环境分 + 行业环境分 + 情绪环境分

大盘环境分 (-5 ~ +5):
- 大盘处于明显熊市: +5分 (逆向投资)
- 大盘震荡下行: +3分
- 大盘震荡: 0分
- 大盘震荡上行: -2分
- 大盘明显牛市: -5分

行业环境分 (-3 ~ +3):
- 行业处于周期底部: +3分
- 行业下行中: +1分
- 行业平稳: 0分
- 行业上行: -1分
- 行业处于周期顶部: -3分

情绪环境分 (-3 ~ +3):
- 市场极度恐慌(VIX高位): +3分
- 市场偏悲观: +1分
- 情绪中性: 0分
- 市场偏乐观: -1分
- 市场极度贪婪: -3分
```

### 3.5 动量确认分

```
动量确认分 (-5 ~ +5):

正向确认 (+1 ~ +5):
- 日线级别MACD底背离: +2分
- 周线级别出现积极信号: +2分
- 多指标同步超卖: +1分

负向确认 (-1 ~ -5):
- 指标持续钝化(超卖后继续下跌): -3分
- 趋势未企稳，继续创新低: -2分
- 成交量持续萎缩无反弹迹象: -2分
```

## 四、综合评分与买入决策矩阵

### 4.1 评分等级划分

| 总评分 | 等级 | 含义 | 定投策略 |
|--------|------|------|---------|
| 85-100 | **S级** | 极度超卖，强烈买入 | 双倍定投或一次性大额加仓 |
| 70-84 | **A级** | 严重超卖，积极买入 | 标准定投 + 额外加仓 |
| 55-69 | **B级** | 中度超卖，适度买入 | 标准定投 |
| 40-54 | **C级** | 轻度超卖，谨慎买入 | 标准定投或减半定投 |
| 25-39 | **D级** | 中性区域，观望 | 暂停定投或小额定投 |
| 0-24 | **F级** | 偏强区域，不买入 | 暂停定投 |

### 4.2 买入决策流程

```
开始
  │
  ▼
计算综合评分
  │
  ├── 评分 ≥ 85 (S级) ──→ 双倍定投 + 考虑一次性加仓
  │
  ├── 评分 70-84 (A级) ──→ 标准定投 + 额外加仓20%
  │
  ├── 评分 55-69 (B级) ──→ 标准定投
  │
  ├── 评分 40-54 (C级) ──→ 标准定投或减半定投
  │
  ├── 评分 25-39 (D级) ──→ 暂停定投或小额定投(500元)
  │
  └── 评分 < 25 (F级) ──→ 暂停定投
  │
  ▼
执行定投后持续监控
  │
  ▼
每周/每月重新评估评分
  │
  ▼
根据评分变化动态调整策略
```

## 五、多标的轮动策略

### 5.1 评分比较法

```
每月定投日：
1. 计算所有8个标的的综合评分
2. 选择评分最高的3个标的进行定投
3. 资金分配：按评分高低加权分配

资金分配公式：
标的i的定投金额 = 2000 × (标的i评分 / 前三名评分总和)
```

### 5.2 行业/风格轮动补充

```
根据市场环境动态调整标的权重：

牛市初期：增加中证A50、中证A500权重
牛市中期：增加中证500、中证1000权重
牛市后期：转向红利低波、中证红利（防御）

熊市初期：全部转向红利低波、东证红利低波
熊市中期：保持红利策略，减少宽基
熊市后期：逐步增加宽基指数权重
```

## 六、风险控制与退出机制

### 6.1 止损规则

```
单笔定投止损：
- 定投后标的下跌超过15%：触发评估
- 定投后标的下跌超过25%：触发止损，停止该标的定投

组合止损：
- 整体组合浮亏超过20%：全面评估策略
- 整体组合浮亏超过30%：考虑减仓至50%
```

### 6.2 止盈规则

```
动态止盈：
- 单个标的盈利超过30%：考虑减仓1/3
- 单个标的盈利超过50%：考虑减仓1/2
- 综合评分进入F级(0-24分)：考虑全部止盈

目标止盈：
- 年化收益率达到15%：可部分止盈
- 年化收益率达到25%：建议大幅止盈
```

### 6.3 再平衡机制

```
每季度再平衡：
1. 检查各标的持仓比例
2. 偏离目标配置超过10%：进行再平衡
3. 根据最新评分调整标的权重
```

## 九、注意事项

1. **指标钝化**：在强趋势行情中，RSI、KDJ可能长期钝化，需结合趋势指标判断
2. **背离确认**：背离信号需等待价格确认，不可仅凭背离就买入
3. **多周期验证**：日线信号需结合周线、月线级别验证
4. **基本面配合**：技术指标需结合基本面分析，避免陷入价值陷阱
5. **资金管理**：严格执行定投纪律，避免情绪化操作
6. **长期视角**：定投策略需要长期坚持，短期波动不影响长期收益
