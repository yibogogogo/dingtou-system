| **更新方式** | 每日自动更新 + 手动刷新 |

## 第二部分：技术指标计算

### 2.1 RSI(6,12,24) — 相对强弱指数

```
RSI = 100 - [100 / (1 + RS)]
RS = n日内平均上涨幅度 / n日内平均下跌幅度
```

| 等级 | RSI(6) | RSI(12) | RSI(24) | 触发分数 |
|------|--------|---------|---------|---------|
| 极度超卖 | <15 | <20 | <25 | +25 |
| 严重超卖 | 15-25 | 20-30 | 25-35 | +20 |
| 中度超卖 | 25-35 | 30-40 | 35-45 | +15 |
| 轻度超卖 | 35-45 | 40-50 | 45-55 | +10 |

### 2.2 KDJ(9,3,3) — 随机指标

```
RSV = (收盘价 - 9日最低) / (9日最高 - 9日最低) × 100
K = 2/3 × 前日K + 1/3 × 当日RSV
D = 2/3 × 前日D + 1/3 × 当日K
J = 3×D - 2×K
```

| 等级 | K值 | D值 | J值 | 触发分数 |
|------|-----|-----|-----|---------|
| 极度超卖 | <10 | <15 | <0 | +25 |
| 严重超卖 | 10-20 | 15-25 | 0-10 | +20 |
| 中度超卖 | 20-30 | 25-35 | 10-20 | +15 |

### 2.3 BIAS(5,10,20,60) — 乖离率

```
BIAS(n) = (收盘价 - n日均价) / n日均价 × 100%
```

| 周期 | 轻度偏离 | 中度偏离 | 严重偏离 | 极度偏离 |
|------|---------|---------|---------|---------|
| BIAS(5) | -2%~-3% | -3%~-5% | -5%~-8% | <-8% |
| BIAS(10) | -3%~-5% | -5%~-8% | -8%~-12% | <-12% |
| BIAS(20) | -5%~-8% | -8%~-12% | -12%~-18% | <-18% |
| BIAS(60) | -8%~-12% | -12%~-18% | -18%~-25% | <-25% |

### 2.4 MACD（辅助确认）

```
EMA12 = 12日指数移动平均
EMA26 = 26日指数移动平均
DIF = EMA12 - EMA26
DEA = DIF的9日指数移动平均
MACD柱 = 2 × (DIF - DEA)
```

**信号**：DIF上穿DEA（金叉）+ MACD柱由负转正 → 买入确认

## 第三部分：综合评分引擎

### 3.1 评分公式

```
总评分 = 技术指标(40) + 估值因子(25) + 动量因子(15) + 情绪因子(10) + 资金因子(10)

满分 = 100
```

### 3.2 技术指标评分规则（满分40分）

```python
def compute_technical_score(rsi6, rsi12, rsi24, k, d, j, bias20, bias60, dif, dea):
    score = 0

    # ===  RSI  ===  (满分15)
    if rsi6 < 15: score += 6
    elif rsi6 < 25: score += 4
    elif rsi6 < 35: score += 2

    if rsi12 < 20: score += 5
    elif rsi12 < 30: score += 3
    elif rsi12 < 40: score += 1

    if rsi24 < 25: score += 4
    elif rsi24 < 35: score += 2
    elif rsi24 < 45: score += 1

    # RSI金叉加分
    if rsi6 > rsi12: score += 2

    # ===  KDJ  ===  (满分15)
    if k < 10: score += 4
    elif k < 20: score += 3
    elif k < 30: score += 1

    if d < 15: score += 4
    elif d < 25: score += 3
    elif d < 35: score += 1

    if j < 0: score += 5
    elif j < 10: score += 3
    elif j < 20: score += 1

    # KDJ金叉
    if k > d: score += 2

    # ===  BIAS  ===  (满分10)
    if bias20 < -15: score += 5
    elif bias20 < -10: score += 3
    elif bias20 < -5: score += 1

    if bias60 < -20: score += 5
    elif bias60 < -15: score += 3
    elif bias60 < -8: score += 1

    # ===  MACD  ===  (加分项)
    if dif > dea: score += 3  # MACD金叉

    return min(score, 40)
```

### 3.3 估值因子评分规则（满分25分）

```python
def compute_valuation_score(pe, pe_percentile, pb, dividend_yield):
    score = 0

    # PE绝对值
    if pe < 6.0: score += 8
    elif pe < 7.0: score += 6
    elif pe < 8.0: score += 4
    elif pe < 9.0: score += 2

    # PE历史分位
    if pe_percentile is not None:
        if pe_percentile < 10: score += 6
        elif pe_percentile < 20: score += 5
        elif pe_percentile < 30: score += 3
        elif pe_percentile < 50: score += 1

    # PB
    if pb < 0.7: score += 6
    elif pb < 0.8: score += 4
    elif pb < 0.9: score += 2

    # 股息率
    if dividend_yield is not None:
        if dividend_yield > 5.5: score += 5
        elif dividend_yield > 5.0: score += 4
        elif dividend_yield > 4.5: score += 3
        elif dividend_yield > 4.0: score += 2

    return min(score, 25)
```

### 3.4 动量因子评分规则（满分15分）

```python
def compute_momentum_score(monthly_return, quarterly_return, yearly_return):
    """
    monthly_return: 近1月涨跌幅（负值为跌）
    quarterly_return: 近1季涨跌幅
    yearly_return: 近1年涨跌幅
    """
    score = 0

    # 月动量：跌幅越大，越有反弹潜力
    if monthly_return < -10: score += 6
    elif monthly_return < -5: score += 4
    elif monthly_return < -2: score += 2
    elif monthly_return < 0: score += 1

    # 季动量
    if quarterly_return < -20: score += 5
    elif quarterly_return < -10: score += 3
    elif quarterly_return < -5: score += 1

    # 年动量
    if yearly_return < -30: score += 4
    elif yearly_return < -15: score += 3
    elif yearly_return < -5: score += 1

    return min(score, 15)
```

### 3.5 情绪因子评分规则（满分10分）

```python
def compute_sentiment_score(fear_index, margin_balance_change, new_accounts_change):
    """
    fear_index: 市场恐慌程度（0-100，越高越恐慌）
    margin_balance_change: 融资余额月度变化率（负值=去杠杆）
    new_accounts_change: 新增开户数变化
    """
    score = 0

    if fear_index > 80: score += 5      # 极度恐慌
    elif fear_index > 60: score += 3    # 偏恐慌
    elif fear_index > 40: score += 1

    if margin_balance_change < -10: score += 3    # 大幅去杠杆
    elif margin_balance_change < -5: score += 2
    elif margin_balance_change < 0: score += 1

    if new_accounts_change < -30: score += 2     # 开户数骤降
    elif new_accounts_change < -10: score += 1

    return min(score, 10)
```

### 3.6 资金因子评分规则（满分10分）

```python
def compute_fundflow_score(north_flow, etf_flow, main_flow):
    """
    north_flow: 北向资金近期流向（负值=流出）
    etf_flow: 对应ETF资金流向
    main_flow: 主力资金流向
    """
    score = 0

    # 北向资金：持续流出后企稳是机会
    if north_flow < -100: score += 4      # 大幅流出
    elif north_flow < -50: score += 2
    elif north_flow < 0: score += 1

    # ETF资金：持续赎回后企稳
    if etf_flow < -50: score += 3
    elif etf_flow < -20: score += 2
    elif etf_flow < 0: score += 1

    # 主力资金
    if main_flow < -50: score += 3
    elif main_flow < -20: score += 2
    elif main_flow < 0: score += 1

    return min(score, 10)
```

### 4.2 动态资金分配

```
Step 1: 计算三个标的的总评分
Step 2: 过滤评分 < 25 的标的
Step 3: 按评分比例分配资金

标的i金额 = 总定投金额 × 倍数 × (标的i评分 / 合格标的评分总和)
```

## 第八部分：关键工程要点

### 8.1 数据更新机制
```
1. 启动时自动检查本地数据最新日期
2. 通过 akshare 增量拉取缺失数据
3. 每周五 15:00 自动触发完整指标重算
4. 每月第一个交易日发出定投提醒
```

### 8.2 性能优化
```
1. 历史日线数据使用 Parquet 格式存储（比CSV快10倍）
2. 技术指标计算结果缓存，仅在数据更新后重算
3. Plotly 图表使用 WebGL 渲染模式处理大量数据点
4. Streamlit 使用 @st.cache_data 缓存评分结果
```

### 8.3 容错处理
```
1. akshare 接口不可用时，回退使用本地缓存数据
2. 数据更新失败时，使用最新可用数据继续运行
3. 单只指数数据异常时，仅标记该指数不可用，不影响其他
```

```bash
pip install streamlit plotly akshare pandas numpy ta schedule

# 初始化数据（首次运行）
python data/fetcher.py --init

# 启动 Dashboard
streamlit run app.py --server.port 8501

# 手动触发数据更新
python data/updater.py --force

# 运行回测
python engine/backtest.py --start 20190101 --end 20260721
```
