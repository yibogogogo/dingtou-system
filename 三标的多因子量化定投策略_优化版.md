## 四、买入决策矩阵

### 4.1 评分等级

| 总评分 | 等级 | 含义 | 定投策略 |
|--------|------|------|---------|
| 85-100 | S级 | 极度超卖，强烈买入 | 双倍定投(4000元) |
| 70-84 | A级 | 严重超卖，积极买入 | 标准定投+20%(2400元) |
| 55-69 | B级 | 中度超卖，适度买入 | 标准定投(2000元) |
| 40-54 | C级 | 轻度超卖，谨慎买入 | 减半定投(1000元) |
| 25-39 | D级 | 中性区域，观望 | 暂停或小额(500元) |
| 0-24 | F级 | 偏强区域，不买入 | 暂停定投 |

### 4.2 三标动态配置

```
每月定投日：
1. 计算三个标的综合评分
2. 筛选评分≥40的标的
3. 按评分比例分配资金

资金分配公式：
标的i定投金额 = 总定投金额 × (标的i评分 / 合格标的评分总和)

示例：
- 科创50评分85，中证红利评分65，红利低波评分55
- 总评分 = 85 + 65 + 55 = 205
- 科创50: 2000 × 85/205 = 830元
- 中证红利: 2000 × 65/205 = 634元
- 红利低波: 2000 × 55/205 = 536元
```

## 五、风险控制

### 5.1 止损规则

| 条件 | 操作 |
|------|------|
| 单标的定投后下跌>20% | 评估基本面，决定是否暂停 |
| 单标的定投后下跌>30% | 暂停该标的定投 |
| 连续3个月评分<25 | 暂停该标的，转入观察 |
| 组合浮亏>15% | 全面评估策略有效性 |
| 组合浮亏>25% | 减仓至50%，保留现金 |

### 5.2 止盈规则

| 条件 | 操作 |
|------|------|
| 单标的盈利>30% | 减仓1/3 |
| 单标的盈利>50% | 减仓1/2 |
| 综合评分进入F级 | 考虑全部止盈 |
| 年化收益达15% | 部分止盈(减仓20%) |
| 年化收益达25% | 大幅止盈(减仓50%) |
| 年化收益达35% | 全部止盈，等待机会 |

## 六、执行流程

```
每月定投日：
Step 1: 获取三个标的最新价格数据
Step 2: 计算RSI(6,12,24)、KDJ(9,3,3)、BIAS(5,10,20,60)
Step 3: 获取估值数据(PE/PB/股息率)
Step 4: 获取市场情绪和资金流向数据
Step 5: 计算各标的综合评分
Step 6: 根据评分确定各标的定投金额
Step 7: 执行定投交易
Step 8: 记录交易日志
Step 9: 评估止盈止损条件
Step 10: 更新持仓跟踪表
```
